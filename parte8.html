Las utilidades más usadas en Tailwind CSS v4.1

Estructura, espaciado, texto, colores, y comportamiento responsive.

En esta parte nos detenemos en las utilidades que más vas a usar cuando empieces a construir interfaces web con Tailwind. La idea es entender bien qué representan, cómo se aplican y cómo se combinan entre sí. Cada bloque incluye ejemplos muy comentados, como si estuviéramos viendo el código juntos por primera vez.

1) Espaciado: padding (p), margin (m), gap, space

El espaciado es una de las bases del diseño con Tailwind. En lugar de escribir CSS como padding: 10px;, usas una escala numérica pensada para el diseño.

  • p-4 → padding completo de 1rem (16px si la raíz es 16px).
  • px-4 → padding horizontal (left + right).
  • pb-6 → padding inferior; 6 equivale a 1.5rem.
  • m-8 → margin completo 2rem.
  • my-4 → margin vertical (top + bottom).
  • gap-4 → separación entre hijos en un flex o grid.
<div
  class="p-6               /* padding en todo el contenedor */
         mt-8              /* margin-top: separa este bloque de lo anterior */
         rounded-xl shadow /* redondeado + sombra */
         bg-white"
>
  <h2 class="mb-4 text-xl font-semibold">Título</h2>
  <p class="mb-6 text-gray-600">Texto con espacio inferior.</p>
  <div class="flex gap-4">
    <button class="px-4 py-2 bg-blue-600 text-white rounded">Aceptar</button>
    <button class="px-4 py-2 bg-gray-200 text-gray-800 rounded">Cancelar</button>
  </div>
</div>

Comentarios:

  • Usamos gap-4 para espaciar botones sin márgenes manuales.
  • mb-4 y mb-6 crean un flujo visual claro.
  • Toda la escala está predefinida, así ganas consistencia entre elementos sin pensar en “cuántos px”.

2) Tamaño: width (w) y height (h)

Controlar el tamaño de los elementos es fundamental. Tailwind ofrece una escala para anchos y altos.

  • w-full → width: 100%.
  • w-1/2 → width: 50%.
  • w-64 → width: 16rem (256px).
  • h-screen → height: 100vh.
  • h-16 → height: 4rem (64px).
<div class="flex gap-4">
  <div class="w-1/2 bg-blue-200 p-4 rounded">50% de ancho</div>
  <div class="w-1/2 bg-green-200 p-4 rounded">50% de ancho</div>
</div>
<div class="h-32 w-full bg-gray-200 mt-4 rounded flex items-center justify-center">
  Alto fijo (h-32) y ancho completo (w-full)
</div>

3) Tipografía: font-size, weight, color

Estilizar texto es muy intuitivo con las utilidades de tipografía.

  • text-lg → font-size: 1.125rem.
  • text-2xl → font-size: 1.5rem.
  • font-bold → font-weight: 700.
  • font-light → font-weight: 300.
  • text-center → text-align: center.
  • leading-relaxed → line-height: 1.625.
  • tracking-wider → letter-spacing: 0.05em.
<h1 class="text-3xl font-bold text-gray-900">Título Principal</h1>
<p class="mt-2 text-lg text-gray-600 leading-relaxed">
  Este es un párrafo con un tamaño de fuente más grande y un interlineado más relajado para facilitar la lectura.
</p>

4) Colores: background, text, border

Tailwind viene con una paleta de colores muy completa y personalizable.

  • bg-blue-500 → background-color: #3b82f6.
  • text-red-600 → color: #dc2626.
  • border-gray-300 → border-color: #d1d5db.
  • La escala va de 50 (más claro) a 900 (más oscuro).
<div class="bg-yellow-100 border-l-4 border-yellow-500 text-yellow-700 p-4" role="alert">
  <p class="font-bold">Alerta</p>
  <p>Esto es un componente de alerta que usa colores para comunicar significado.</p>
</div>

5) Flexbox y Grid

Para alinear elementos, Flexbox y Grid son esenciales.

  • flex → display: flex.
  • grid → display: grid.
  • items-center → align-items: center.
  • justify-between → justify-content: space-between.
  • grid-cols-3 → grid-template-columns: repeat(3, minmax(0, 1fr)).
<div class="flex items-center justify-between p-4 bg-gray-100 rounded-lg">
  <div>Logo</div>
  <div class="flex gap-4">
    <a href="#">Enlace 1</a>
    <a href="#">Enlace 2</a>
  </div>
</div>

<div class="grid grid-cols-1 sm:grid-cols-3 gap-4 mt-4">
  <div class="bg-blue-200 p-4 rounded">Col 1</div>
  <div class="bg-green-200 p-4 rounded">Col 2</div>
  <div class="bg-red-200 p-4 rounded">Col 3</div>
</div>

6) Diseño Responsivo (Breakpoints)

Aplica utilidades condicionalmente en diferentes tamaños de pantalla.

  • sm: → min-width: 640px.
  • md: → min-width: 768px.
  • lg: → min-width: 1024px.
  • xl: → min-width: 1280px.
<div class="bg-red-500 md:bg-green-500 lg:bg-blue-500 p-4 rounded-lg text-white">
  El fondo de este div cambia de color según el ancho de la pantalla.
  <ul>
    <li>Rojo en pantallas pequeñas.</li>
    <li>Verde en pantallas medianas (md).</li>
    <li>Azul en pantallas grandes (lg).</li>
  </ul>
</div>

<div class="text-center lg:text-left mt-4">
  Texto centrado en móvil, alineado a la izquierda en pantallas grandes.
</div>
